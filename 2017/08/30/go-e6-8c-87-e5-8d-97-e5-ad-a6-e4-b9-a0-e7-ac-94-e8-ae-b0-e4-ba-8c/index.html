Go 指南学习笔记二

发表于 2017-08-30 | 分类于 golang

1.for循环

go语言只有一个循环语句，即for。如下

sum := 0

for i:= 0;i&lt;10;i++{


&nbsp; &nbsp; sum += i


}
  • 不像java,python等，条件表达式不需要()小括号，循环体需要{}
  • 初始化条件和后置判断条件不是必须的，如for ; i<10; {}
  • 在java等语言中while循环在go语言中写法

for i< 10 {

&nbsp; &nbsp; sum += i


}
  • 死循环

for {

&nbsp; &nbsp; sum += i


}

2.if语句
  • 同for循环一样,if语句的判断条件语句也不需要()

if i<10 {

}
  • 同for循环一样，条件之前也可以是一个简单的语句

i := 10

if j:= 10; i&lt;j {


&nbsp; &nbsp; return j


}
  • else

i := 10

if j := 10; i&lt;j {


&nbsp; &nbsp; return j


} else {


&nbsp; &nbsp; return i


}

3.switch语句

&nbsp;

&nbsp; &nbsp; case &quot;darwin&quot;:


&nbsp; &nbsp; &nbsp; &nbsp; fmt.Println(&quot;OS X.&quot;)

  • 没有条件的switch，如switch {},同switch true {}一样

    4.defer语句

  • defer 语句会延迟函数的执行直到上层函数返回。

    延迟调用的参数会立刻生成，但是在上层函数返回前函数都不会被调用

package main

import &quot;fmt&quot;


func main(){


&nbsp; &nbsp; defer fmt.Println(&quot;world&quot;)


&nbsp; &nbsp; fmt.Print(&quot;hello&quot;)


}
  • defer栈，延迟的函数调用会被压入一个栈中。
