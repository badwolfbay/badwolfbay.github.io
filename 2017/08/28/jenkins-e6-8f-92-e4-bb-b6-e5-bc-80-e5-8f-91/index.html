3.创建新的插件，在命令行执行
mvn -U org.jenkins-ci.tools:maven-hpi-plugin:create
过程中要求输入插件的groupId和artifactId

编译新建的插件

mvn install
 

4.本地调试：

set MAVEN_OPTS=-Xdebug -Xrunjdwp:transport=dt_socket,server=y,address=8000,suspend=n
 
mvn hpi:run
